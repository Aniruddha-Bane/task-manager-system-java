# Task Manager System (Java)

A console-based Task Manager application built using Core Java.

This project demonstrates real-world backend development concepts such as CRUD operations, object-oriented programming, workflow management, reporting, file persistence, and layered architecture.

---

## Features

- Add Task
- View Tasks
- Search Task by ID
- Update Task Status
- Delete Task
- Generate Reports
- File Persistence (tasks.txt)
- Menu Driven Interface

---

## Technologies Used

- Java
- OOP
- ArrayList
- File Handling
- Scanner
- IntelliJ IDEA
